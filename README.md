# 三路超声波智能避障小车

基于 STM32F103C8T6 的自主避障小车，通过三路超声波传感器实现全方位障碍物检测与智能路径规划。

## 硬件平台

| 组件 | 型号/规格 |
|------|----------|
| 主控 | STM32F103C8T6（72MHz） |
| 超声波 | HC-SR04 × 3（前/左/右，45°安装） |
| 电机 | 直流减速电机 × 2 |
| 驱动 | GPIO + PWM 方向/速度控制 |
| 编码器 | 光电编码器 × 2（脉冲+方向） |
| 陀螺仪 | MPU6050（I2C，预留） |
| 显示 | 0.96寸 OLED（软件I2C，预留） |
| 蜂鸣器 | 有源蜂鸣器，PB12，低电平触发 |
| 调试 | USART1，9600bps |

| 右侧超声 Echo | PB9 | EXTI9 |

| 左侧超声 Echo | PB3 | EXTI3（需禁JTAG） |
| 编码器1 | PA2 / PA3 | 脉冲 / 方向 |
| 编码器2 | PB5 / PB6 | 脉冲 / 方向 |

## 中断分配

| 中断 | 用途 |
|------|------|
| EXTI2 | 编码器1脉冲计数 |
| EXTI3 | 左侧超声波Echo |
| EXTI9_5 | 右侧超声Echo + 编码器2脉冲 |
| EXTI15_10 | 正前方超声Echo |
| TIM2 | 电机PWM输出 |
| TIM4 | 超声波计时（1MHz自由运行） |
| USART1 | 串口接收 |

## 避障策略

```
检测到障碍
    │
    ├─ 任意方向极近（<8cm / <5cm） → 紧急后退
    │
    ├─ 前方有障碍（<20cm）→ 比较左右空间，向更开阔侧转弯
    │
    ├─ 左侧有障碍（<12cm）→ 右转
    │
    ├─ 右侧有障碍（<12cm）→ 左转
    │
    ├─ 前方中距离 + 左侧极开阔 → 弯道探索（左转）
    │
    └─ 无障碍 → 直行
```

### 出口检测（波谷法）

用于检测小车是否穿过狭窄出口（如门口）。原理：

1. 持续追踪左右两侧距离的局部最小值（波谷）
2. 当两侧当前值都回升到阈值以上，且之前的波谷都曾低于阈值 → 判定通过出口
3. 触发后进入冷却期，防止重复检测

## 速度与阈值配置

所有可调参数集中在 `USER/speed.h`，无需修改其他文件：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `SPEED_GO_L/R` | 150 | 直行速度（0~899） |
| `SPEED_TURN_L/R` | 150 | 转弯速度 |
| `SPEED_BACK_L/R` | 140 | 后退速度 |
| `THR_HEAD_EMRG` | 8 | 前方紧急距离（cm） |
| `THR_SIDE_EMRG` | 5 | 侧向紧急距离 |
| `THR_HEAD_NEAR` | 20 | 前方障碍距离 |
| `THR_LEFT/RIGHT_NEAR` | 12 | 左右侧障碍距离 |
| `DELAY_TURN` | 50ms | 转弯持续时间 |

## 项目结构

```
code515/project/
├── CORE/               # CMSIS核心文件
├── STM32F10x_FWLib/    # STM32标准外设库
├── SYSTEM/             # 系统级驱动（delay/usart/sys）
├── HARDWARE/           # 外设驱动模块
│   ├── GPIO/           # 引脚初始化 + 电机控制
│   ├── PWM/            # TIM2 PWM输出
│   ├── TIMER/          # TIM4自由运行 + 编码器EXTI
│   ├── READ/           # 超声波测距（EXTI中断捕获）
│   ├── AI/             # 避障决策算法
│   ├── MPU6050/        # 陀螺仪驱动（I2C）
│   ├── OLED/           # OLED显示（软件I2C）
│   ├── I2C/            # 硬件I2C驱动
│   └── BUZZER/         # 蜂鸣器控制
└── USER/
    ├── main.c          # 主程序
    ├── speed.h         # 速度与阈值参数
    └── stm32f10x_it.c  # 中断服务函数
```

## 开发环境

- **IDE：** Keil MDK-ARM
- **调试器：** J-Link（SWD模式）
- **编译器：** ARM Compiler v5
- **串口工具：** 任意串口助手，波特率9600

## 使用说明

1. 用 Keil 打开 `USER/PWM.uvprojx`
2. 编译无误后通过 J-Link 烧录
3. 上电后蜂鸣器响一声表示初始化完成
4. 小车自动进入避障模式
5. 串口输出实时距离和PWM数据，可用于调试

## 可选模块

### 启用MPU6050

在 `main.c` 中取消注释：

```c
#include "i2c.h"
#include "mpu6050.h"

// 在main()中：
I2C2_Init();
MPU6050_Init();
```

### 启用OLED显示

在 `main.c` 中取消注释：

```c
#include "oled.h"

// 在main()中：
OLED_Init();
```

> ⚠️ OLED与右侧超声波共用PB8/PB9，显示时会影响该路测距，建议仅在调试阶段使用。
